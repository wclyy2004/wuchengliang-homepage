# 吴成亮个人网站

这是一个纯静态个人学术网站，可直接双击 `index.html` 在浏览器中打开，也可以上传到学校服务器、GitHub Pages 或任意静态网站空间。

## 文件说明

- `index.html`：页面内容
- `styles.css`：页面样式
- `script.js`：导航栏滚动效果与年份
- `assets/hero-biomechanics.png`：首屏运动生物力学视觉图
- `assets/concept.png`：设计概念参考图，不在页面中直接使用

## 后续建议

1. 如需加入个人照片，可将照片保存为 `assets/portrait.jpg`，并把 `index.html` 中的 `.portrait-placeholder` 区域替换为图片。
2. 公开发布前，建议核对职称、项目、论文卷期页码和联系方式。
3. 若用于学校官网，可删除页脚中的“公开发布前建议核对……”提示语。
